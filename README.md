# Qt_officeCooperation
Qt大作业，办公协作软件，计划实现注册、登录、即时通讯、文件互传、文本编辑、日程表等功能

其中
- officecooperation文件夹存放了用户电脑运行的代码（win11 Qt5.15.2 需要加载MySQL数据库驱动）
- Server文件夹存放服务器上运行的代码，用来监听用户的连接，转发消息，文件（centos7.9 Qt5.12.0）

**已经实现的功能**
- 注册
- 登录
- 添加联系人
- 删除联系人
  - 右键删除
- 搜索已有联系人
- 点击头像，个人页面设置
  - 头像
  - 背景
  - 密码
- 设定计划
- 待办事项（to do list）
  - 左键点击完成
- 项目编辑
  - 文本编辑
  - 项目管理
- 发送消息
  - 左键打开聊天窗口
- 发送文件
